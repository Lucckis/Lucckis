## Sobre mim
Meu nome é Lucas, tenho 19 anos e curso Análise e Desenvolvimento de Sistemas na FIAP. Sou um desenvolvedor focado em soluções tanto Front-End como Banco de Dados, além de treinar modelos IA utilizando bibliotecas em python também, com noções básicas de Back-End e experiência na criação de aplicações robustas e escaláveis.
 
Meu objetivo é construir sites e aplicativos bem estruturados, e aplicações robustas utilizando:
- **Front-End:** React.js e React Native
- **Back-End:** Java (Spring Boot) / Python
- **Banco de Dados:** SQL / PL/SQL (Oracle, MySQL, Postgres) / MongoDB
- **Arquitetura:** APIs REST e Microserviços
- **Infraestrutura:** Containerização com Docker
- **Cloud:** Deploy e serviços em nuvem com Azure
